⚙️ Come configurare Firebase

  1. Vai su firebase.google.com e accedi con il tuo account Google
  2. Clicca Aggiungi progetto → dai un nome (es. pocchiaciclopedia) → continua
  3. Nel progetto: menu laterale → Realtime Database → Crea database → scegli Modalità test → Abilita
  4. Menu laterale → Impostazioni progetto (ingranaggio) → tab Generali → scorri giù → Aggiungi app → icona Web </>
  5. Dai un nome all'app (qualsiasi) → Registra app
  6. Copia il blocco firebaseConfig = { ... } che appare
  7. Apri index.html in un editor di testo → cerca FIREBASE_CONFIG → incolla i valori al posto dei segnaposto INSERISCI_…
  8. Salva e ricarica su GitHub → la wiki sarà condivisa con tutti 🎉
Piano gratuito Firebase (Spark): 1 GB di dati, 50.000 letture/giorno, 20.000 scritture/giorno — più che sufficiente per la Pocchiaciclopedia. Nessun costo.
